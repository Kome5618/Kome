# ままのスライド画像生成プロンプトai 🦸‍♀️  
Busy moms' SNS slide generator

忙しいママのSNS投稿をサポートする  
スライド投稿用プロンプト生成ツールです。

スタイル・ターゲット・スライド内容を入力すると  
Gemini / ChatGPT に貼るだけの画像生成プロンプトを自動で作成します。

🌍 Live Demo
👉 https://kome5618.github.io/Kome/
（GitHub Pages公開）

✨ Features
- SNSスライド投稿プロンプトを自動生成
- ターゲット別テンプレート
- 情景サンプル自動生成
- Before / After 比較スライド
- テキストスライド / ビジュアルスライド対応
- Threads / Instagram サイズ対応
- スタイル選択

🎨 Available Styles
- 3Dクレイ
- ゆるっとイラスト
- ミニマルイラスト
- コミック
- テキスト
- ノート風まとめ

スタイルを変えると  
画像生成の雰囲気が変わります。

 👩‍👧 Target Presets
ターゲットを選ぶと
- 入力例
- 情景サンプル
- プロンプト方向
が自動で変わります。

現在のターゲット
- 育児ママ
- 美容系ママ
- 副業ママ

🚀 How to use

1️⃣ スタイルを選ぶ  
2️⃣ ターゲットを選ぶ  
3️⃣ スライド内容を入力  
4️⃣ プロンプト生成  
5️⃣ Gemini / ChatGPT に貼る!!メインキャラがいる場合は画像も一緒にアップロードすること！！

画像生成AIで  
そのままスライド画像が作れます。

🧠 Why this tool exists
SNS投稿を作るとき
- 構成を考える  
- 情景を書く  
- プロンプトを書く  
これに時間がかかります。
このツールは  
それを数秒で作れるようにするためのものです。
